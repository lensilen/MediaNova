# User Manual - MediaNova

Panduan ini ditujukan untuk pengguna akhir MediaNova.

## 1. Login dan Register

1. Buka aplikasi MediaNova.
2. Pilih Register jika belum punya akun.
3. Isi nama, email, dan password.
4. Password harus cukup aman dan tidak kosong.
5. Jika sudah punya akun, pilih Login.
6. Pengguna juga dapat login menggunakan Google Sign-In pada build Android.

Screenshot: `assets/screenshots/01-login.png`

## 2. Melihat Feed

1. Setelah login, pengguna masuk ke halaman Home.
2. Feed menampilkan video, foto, dan audio dari user lain.
3. Swipe ke atas/bawah untuk pindah post.
4. Tap video/audio untuk play atau pause.
5. Gunakan tombol kanan untuk like, comment, save, atau share.

Screenshot: `assets/screenshots/02-feed.png`

## 3. Membuat Media Baru

1. Tekan tab Create di navbar bawah.
2. Pilih mode Photo, Video, atau Audio.
3. Untuk video/foto, izinkan akses kamera.
4. Untuk audio, izinkan akses mikrofon.
5. Gunakan tombol record/capture.
6. Setelah selesai, aplikasi menampilkan preview.

Screenshot: `assets/screenshots/03-create-video.png`

## 4. Mengedit Media

1. Dari preview hasil capture, tekan Next.
2. Pada editor, pengguna dapat memilih tool bawah.
3. Video mendukung trim range UI, split UI, volume, speed, beauty, text, dan filter.
4. Foto mendukung filter, sticker, beauty, dan text overlay.
5. Audio mendukung editor sederhana seperti trim, volume, speed, dan preview waveform.
6. Setelah selesai, tekan Next untuk lanjut ke New Post.

Screenshot: `assets/screenshots/04-editor.png`

## 5. Membuat Post

1. Isi title dan caption.
2. Tambahkan location bila perlu.
3. Atur siapa yang dapat melihat post.
4. Aktifkan/nonaktifkan komentar.
5. Tekan Post.
6. Tunggu upload sampai 100%.
7. Setelah berhasil, post masuk ke feed utama dan profile.

Screenshot: `assets/screenshots/05-new-post.png`

## 6. Komentar

1. Tekan ikon comment pada feed.
2. Tulis komentar.
3. Tekan kirim.
4. Komentar akan muncul pada post dan tercatat di aktivitas profile.

Screenshot: `assets/screenshots/07-comments.png`

## 7. Profile

1. Tekan tab Profile.
2. Profile menampilkan bio, lokasi, statistik, followers, following, dan post.
3. Tab Media berisi video, foto, dan audio yang diupload.
4. Tab Comments berisi post yang pernah dikomentari.
5. Tab Likes berisi post yang disukai.
6. Tab Saved berisi post yang disimpan.
7. Followers dan Following dapat diklik untuk melihat list user.

Screenshot: `assets/screenshots/06-profile.png`

## 8. Search

1. Tekan ikon search di kanan atas Home.
2. Cari user atau konten.
3. Tekan user untuk membuka profile.

Screenshot: `assets/screenshots/09-search.png`

## 9. Notification

1. Tekan ikon notification di kiri atas Home.
2. Aplikasi menampilkan aktivitas seperti follow, like, comment, save, atau mention.
3. Notifikasi dari aksi sendiri tidak ditampilkan agar tidak spam.

Screenshot: `assets/screenshots/08-notifications.png`

## 10. Logout

1. Buka Profile.
2. Masuk ke Settings.
3. Pilih Logout.

## Catatan Pengguna

- Pastikan koneksi internet aktif saat upload media.
- Jika upload gagal, coba ulang dengan file yang lebih kecil.
- Untuk demo notifikasi like/comment/follow, gunakan dua akun berbeda.
