# Screen Recording Demo Script - MediaNova

Durasi target: 3-4 menit.

## 0:00 - 0:20 Opening

- Tampilkan splash screen dan halaman login.
- Jelaskan singkat: MediaNova adalah social media multimedia untuk video, foto, dan audio.

## 0:20 - 0:45 Login/Register

- Login menggunakan akun demo.
- Tunjukkan validasi password/email secara singkat.

## 0:45 - 1:20 Feed Utama

- Tampilkan feed video/foto/audio.
- Swipe post.
- Play/pause video/audio.
- Tunjukkan like, comment, save, dan share.

## 1:20 - 1:55 Create Media

- Masuk ke tab Create.
- Tunjukkan mode video, foto, dan audio.
- Tunjukkan timer, flash, filter warna, sticker overlay, dan record audio waveform.

## 1:55 - 2:35 Editor

- Tunjukkan editor video/foto/audio.
- Pilih tool Trim, Speed, Volume, Beauty, Text, dan Filter.
- Geser text overlay.
- Jelaskan bahwa editor adalah editor sederhana untuk kebutuhan tugas.

## 2:35 - 3:05 Upload Post

- Isi title/caption.
- Tekan Post.
- Tunjukkan progress upload.
- Setelah sukses, post muncul di feed utama.

## 3:05 - 3:35 Profile dan Social

- Buka Profile.
- Tunjukkan Media, Comments, Likes, Saved.
- Tunjukkan Followers/Following.
- Klik salah satu user dari feed/profile.

## 3:35 - 4:00 Search dan Notification

- Tunjukkan search user/content.
- Tunjukkan notification sheet.
- Tutup demo dengan kesimpulan fitur utama.

## Catatan Recording

- Gunakan akun kedua untuk merekam notifikasi like/comment/follow.
- Siapkan minimal 1 video, 1 foto, dan 1 audio sebelum recording.
- Pastikan Cloudinary/Firebase env aktif agar media tampil dari URL.
